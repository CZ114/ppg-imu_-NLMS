## `SX.pkl` 结构（实测 S1，总时长 9212 s）

```
S1.pkl  →  dict {
  'subject':       'S1'                                    # 字符串
  'questionnaire': {AGE:34, Gender:'m', HEIGHT:182,
                    WEIGHT:78, SKIN:3, SPORT:6}
  'activity':      shape (36848, 1)  @ 4 Hz                # 活动 ID 0–8
  'label':         shape (4603,)                           # 心率真值 (BPM)
  'rpeaks':        shape (11431,)                          # ECG R 峰索引
  'signal': {
    'chest': {                                             # RespiBAN @ 700 Hz
      'ACC':  (6448400, 3)    ← 胸带 3 轴加速度
      'ECG':  (6448400, 1)    ← 心电（HR 真值来源）
      'Resp': (6448400, 1)    ← 呼吸
      'EMG','EDA','Temp':     ← dummy，忽略
    }
    'wrist': {                                             # Empatica E4
      'ACC':  (294784, 3)    @ 32 Hz   ← 腕部 3 轴 (单位 1/64 g)
      'BVP':  (589568, 1)    @ 64 Hz   ← PPG 原始波形
      'EDA':  (36848, 1)     @ 4 Hz
      'TEMP': (36848, 1)     @ 4 Hz
    }
  }
}
```

## 各 feature 含义

| Key | 物理意义 | fs | 单位 | NLMS 用？ |
|---|---|---|---|---|
| `signal/wrist/BVP` | **PPG 原始波形** | 64 Hz | a.u. | ✅ 输入 |
| `signal/wrist/ACC` | 腕部加速度 X/Y/Z | 32 Hz | 1/64 g | ✅ 参考噪声 |
| `signal/chest/ECG` | 心电图 | 700 Hz | mV | 真值溯源 |
| `rpeaks` | R 峰样本索引 | — | idx (÷700→s) | 真值溯源 |
| `label` | 8s 窗均 HR | 1/2 s | **BPM** | ✅ 真值 |
| `activity` | 活动 ID | 4 Hz | 类别 | 分活动评估 |
| `signal/chest/ACC` | 胸带加速度 | 700 Hz | g | 一般不用 |
| `signal/chest/Resp` | 呼吸阻抗 | 700 Hz | a.u. | 呼吸频率参考 |
| `questionnaire` | 性别/年龄/肤色/运动频率等 | — | — | 误差分组分析 |

## 活动 ID 映射

| ID | 活动 | 时长 (典型) | 运动伪影强度 |
|---|---|---|---|
| 0 | 过渡/休息 | 不定 | 弱-中 |
| 1 | 静坐阅读 | 10 min | **无**（基线） |
| 2 | 上下楼 | 5 min | **强** |
| 3 | 桌上足球 | 5 min | 中-强 |
| 4 | 户外骑车 | 8 min | 中 |
| 5 | 开车 | 15 min | 弱-中（路面颠簸） |
| 6 | 午餐 | 30 min | 弱（手部动作） |
| 7 | 步行 | 10 min | **强**（周期性） |
| 8 | 办公 | 20 min | 弱 |

## 时序对齐校验公式

对任意被试，记总时长为 $T$ 秒：

- `BVP` 长度 = $64 T$
- `wrist/ACC` 长度 = $32 T$
- `chest/*` 长度 = $700 T$
- `activity` 长度 = $4 T$
- `label` 长度 = $\lfloor (T - 8) / 2 \rfloor + 1$
- 第 $i$ 个 label 对应 BVP 区间 `[i*128 : i*128 + 512]`，即第 $2i$ 到 $2i+8$ 秒
